How It Works

Three steps to test your site

1

Paste your URL

Enter any public website URL into the tool — your live site, a staging environment, or any URL you want to inspect.

2

Choose viewports

Select from preset device sizes — iPhone, Galaxy, iPad, MacBook — or set a fully custom width and height.

3

Inspect side by side

All selected viewports render together so you can immediately spot layout breaks, overflows, or misaligned elements.
